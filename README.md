# ROSE Corp Official Website

Professional website for ROSE Corp - Resilience Research & Translation Infrastructure.

## Overview

This website presents the Rose Glass framework, a mathematical model for understanding and measuring human coherence under systemic misperception.

## Features

- **Responsive Design**: Mobile-first, works beautifully on all devices
- **Modern Stack**: Pure HTML/CSS, no build process required
- **Fast Loading**: Optimized for performance with minimal dependencies
- **SEO Optimized**: Proper meta tags and semantic HTML

## Framework Highlights

- **Four Dimensions**: Ψ (Psi), ρ (Rho), q (Q), f (F)
- **Complete Resilience Model**: Decay + τ-attenuation + restoration
- **Research Foundation**: Based on WP-2026-001 (Revised)
- **Real-world Applications**: Legal, clinical, cultural, educational contexts

## Deployment

### Vercel (Recommended)
1. Push to GitHub
2. Import repository at [vercel.com/new](https://vercel.com/new)
3. Deploy (auto-configured for static site)

### Custom Domain
After Vercel deployment, configure DNS:
```
A Record: @ → 76.76.21.21
CNAME: www → cname.vercel-dns.com
```
